## Description

This collection provides tasks for:

- Linux operating systems:
  - Debian 10/11
  - Ubuntu 20.04/22.04

## Minimum required Ansible-version

- Ansible >= 2.15.0

## Installation

Install the collection via ansible-galaxy:

`ansible-galaxy collection install alphanodes.tasks`
